# 딥러닝 기반 IOT 택배 도난 감지 시스템

## 개요 

 해당 프로젝트는 딥러닝 기술과 IOT 기술을 이용하여 택배 도난 감지 시스템을 개발하는 것이다. 택배 도난 감지 기기에서 도난 여부를 판단할 수 있도록 개발 하였으며, 사용자는 택배 모니터링 앱을 통해 서비스를 이용하므로 사용에 편리하도록 고안되었다.

## 환경 구성

### 		개발 및 운영환경

#### 				택배도난 감지 기기

| 개발 환경 | 규격                     |
| --------- | ------------------------ |
| 편집툴    | Atom, Visual Studio Code |

| OS        | Raspbian 5.4.72          |
